# Athlete Earnings Exploratory Data Analysis

## Project Overview
This project explores earnings patterns of top athletes across sports, countries, and time using exploratory data analysis (EDA).

## Key Insights
- Earnings are heavily right-skewed due to extreme superstar outliers.
- Basketball dominates appearances, while boxing produces the highest single-year earnings.
- Athlete earnings have grown significantly over time.
- Rankings alone do not fully explain earnings differences.

## Tools Used
Python, Pandas, Matplotlib, Seaborn
